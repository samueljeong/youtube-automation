## 3. 세계관

### 3.1 아르테시아 대륙

```

                    ████████████████

    │                                         │
[서부]                 [중앙]                [동부]
드워프 산맥         레온하르트 제국         엘프 숲
    │                    │                    │
    │              ┌─────┴─────┐              │
    │              ↓           ↓              │
    │         [남서부]     [남동부]           │
    │        카렌 왕국    자유도시연합        │
    └─────────────────────────────────────────┘
                    [남부 - 바다]
```

### 3.2 국가 설정

#### 레온하르트 제국 (중앙)
- 체제: 황제 중심 제국
- 특징: 대륙 최강, 소드마스터 15명 보유
- 현황: 마왕군과 최전선 대치
- 분위기: 권위적, 관료주의

#### 카렌 왕국 (남서부)
- 체제: 기사 왕국
- 특징: 기사도 중시, 소드마스터 5명
- 현황: 제국의 동맹국
- 분위기: 정의롭지만 보수적
- **카이든의 출신지**

#### 자유도시연합 (남동부)
- 체제: 상인 길드 연합 (공화제)
- 특징: 모험가 길드 본부, 돈이 곧 법
- 현황: 중립, 모든 국가와 교역
- 분위기: 자유로움, 혼란스러움
- **무영이 정착하는 곳**

#### 엘프 영역 (동부 숲)
- 체제: 장로 회의
- 특징: 폐쇄적, 대마법사 다수
- 현황: 인간과 교류 최소화
- 분위기: 고고함, 순혈주의
- **에이라가 쫓겨난 곳**

#### 드워프 산맥 (서부)
- 체제: 씨족 연합
- 특징: 최고의 대장장이들, 광물 독점
- 현황: 중립, 무기 판매로 부유
- **볼드릭의 출신지**

#### 마왕군 영역 (북방)
- 지배자: **혈마**
- 구성: 마족 + 타락한 인간 + 언데드
- 특징: 흑마법, 마나 오염 지역
- 현황: 세력 확장 중

### 3.3 종족

|------|------|------|
| 인간 | 가장 많음, 적응력 | 80~100년 |

### 3.4 경지 체계

#### 검술 (소드)

| 경지 | 무협 대응 | 대륙 내 인원 | 특징 |
|------|-----------|-------------|------|
| 일반 기사 | 후천지경 | 수십만 명 | 병사, 용병 |
| 엑스퍼트 | 선천지경 | 수천 명 | 기사단 정예 |
| 그래듀에이트 | 절정 | 300~500명 | 기사단장급 |
| **소드마스터** | 화경 | **30~50명** | 국가 전력 |
| **그랜드 소드마스터** | 현경 | **5명 이하** | 대륙 최강 |
| 소드 엠퍼러 | 생사경 | 현재 0명 | 전설 |

#### 마법 (서클)

| 7~8서클 | 수백 명 | 마스터급 |

| 등급 | 대륙 내 | 전투력 |
|------|--------|--------|

### 3.5 핵심 설정: 내공 + 마나 융합

> 이 설정이 무영이 사기 성장하는 핵심 원리

| 항목 | 설명 |
|------|------|
| 마나 = 기(氣) | 본질은 같지만 농도가 다름 |
| 아르테시아 마나 농도 | 무림의 **10배** |
| 일반인의 한계 | 마나 자연 축적 (느림) |
| 무영의 치트 | 심법으로 마나 강제 순환 (빠름) |
| 결과 | 비정상적 성장 속도 |

**왜 이세계 사람들은 심법이 없나?**
- 마나가 풍부해서 자연 축적으로도 충분
- 심법을 연구할 필요성을 못 느낌
- 무영만 아는 비기

## 4. 캐릭터

### 4.1 주인공: 무영

| 항목 | 설정 |
|------|------|
| 이름 | 무영 (이세계에서는 "무"로 불림) |
| 나이 | 20대 초반 |
| 외모 | 동양인 청년, 검은 머리, 날카로운 눈 |
| 전이 상태 | 몸 그대로, **내공만 소실** |
| 남은 것 | 검술 기억 + 심법 지식 + 전투 경험 |

#### 성격

| 특성 | 설명 |
|------|------|
| 쿨함 | 감정 표현 적음, 담담 |
| 어두움 | 과거의 상처, 잘 웃지 않음 |
| 아싸 | 잘 어울리지 못함, 혼자 있는 걸 선호 |
| 의리 | **자기 사람에게는 목숨을 검** |

#### 말투 예시

```
"..."
"시끄럽다."
"상관없어."
"네가 알 필요 없다."
(친구에게) "...고맙다."
(위기 시) "내 뒤에 있어."
```

#### 성장 곡선

| 부 | 경지 |
|----|------|
| 1부 끝 | 그래듀에이트 |
| 2부 끝 | 소드마스터 |
| 4부 | 소드마스터 상위 |
| 6부 | 그랜드 소드마스터 |

### 4.2 히로인: 에이라

| 항목 | 설정 |
|------|------|
| 이름 | 에이라 (Eira) |
| 종족 | 하프엘프 |
| 나이 | 외모 20대 초반 (실제 50대) |
| 직업 | 모험가 + 마법사 |
| 외모 | 은발, 뾰족한 귀, 차가운 미모 |

#### 배경

- 인간 아버지 + 엘프 어머니
- 엘프 영역에서 **혼혈 차별**로 쫓겨남
- 인간 세계에서도 완전히 섞이지 못함
- 혼자 모험가로 활동

#### 성격

| 특성 | 설명 |
|------|------|
| 경계심 | 처음엔 무영도 경계 |
| 외로움 | 어디에도 속하지 못함 |
| 강인함 | 차별에 굴하지 않음 |
| 따뜻함 | 마음 열면 헌신적 |

#### 무영과의 관계

```
1단계: 경계 (둘 다 아웃사이더)
2단계: 공감 (서로의 고독을 이해)
3단계: 교환 (마법 ↔ 심법)
4단계: 신뢰 (서로를 지켜줌)
5단계: 사랑? (설하와의 관계는?)
```

#### 성장 곡선

| 부 | 경지 |
|----|------|
| 첫 등장 | 6서클 |
| 2부 끝 | 7서클 |
| 4부 | 8서클 |
| 5부 | **9서클 (대마법사)** |

### 4.3 빌런: 혈마

| 항목 | 설정 |
|------|------|
| 원래 정체 | 천마교 교주 |
