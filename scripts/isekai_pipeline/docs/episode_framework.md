# 혈영 이세계편 - 에피소드별 플롯 프레임워크

> **이 문서의 목적**: 대본 작성 전 반드시 확인하여 스토리 이탈 방지
>
> **절대 규칙**: 이 프레임워크에 없는 이벤트를 임의로 추가하지 않는다

---

| 캐릭터 | 첫 등장 | 비고 |
|--------|---------|------|
| 무영 | 1화 | 주인공 |
| 카이든 | 2화 | 변방 도시에서 만남 |
| 에이라 | 12화 | 2부 시작 |
| 볼드릭 | 15화 | 검 제작 의뢰 |

## 1부: 이방인 (1~10화)

### 핵심 테마
- 이세계 적응
- 힘 소실과 재기
- 첫 친구 카이든

### 에피소드별 플롯

#### EP001: 이세계 도착
- **시작**: 무림 최종전 회상 (30초 몽타주)
- **핵심 이벤트**:
  1. 낯선 숲에서 눈을 뜸
  2. 내공 소실 자각 (단전이 텅 빔)
  3. 언어 불통 경험
  4. 생존을 위한 첫 행동
- **끝**: 어딘가로 향하며 결심
- **캐릭터 경지**: 일반인 수준 (내공 없음)
- **등장인물**: 무영만

#### EP002: 카이든
- **시작**: EP001 연결 (이동 중)
- **핵심 이벤트**:
  1. 변방 도시 도착
  2. 성문에서 문제 발생
  3. 카이든의 도움
  4. 언어 학습 시작
- **끝**: 카이든과 친구가 됨
- **캐릭터 경지**: 일반인 (내공 없음)
- **등장인물**: 무영, 카이든

#### EP003: 자유도시로
- **시작**: 카이든과 함께 출발
- **핵심 이벤트**:
  1. 여행 중 대화 (세계관 설명)
  2. 이 세계의 강자들에 대해 들음
  3. 자유도시 도착
  4. 모험가 길드 존재 인지
- **끝**: 길드 앞에서 결심
- **등장인물**: 무영, 카이든

#### EP004: 모험가 길드
- **시작**: 길드 입장
- **핵심 이벤트**:
  1. 길드 등록
  2. 등급 시스템 이해
  3. 낮은 평가 (내공 없어서)
  4. 첫 의뢰 선택
- **끝**: 첫 의뢰 출발
- **등장인물**: 무영, 카이든, 길드 직원(단역)

#### EP005: 첫 의뢰
- **시작**: 의뢰 수행 중
- **핵심 이벤트**:
  1. 몬스터 조우
  2. 검술 기억은 살아있음 확인
  3. 하지만 체력/힘 부족
  4. 간신히 의뢰 완료
- **끝**: 한계를 절감
- **등장인물**: 무영

#### EP006: 마나의 발견
- **시작**: 의뢰 후 휴식 중
- **핵심 이벤트**:
  1. 이 세계의 '기'가 마나임을 깨달음
  2. 마나 농도가 무림보다 높음
  3. 심법으로 마나 흡수 시도
  4. 첫 시도 실패
- **끝**: 실마리를 잡음
- **캐릭터 경지**: 마나 감지 시작
- **등장인물**: 무영

#### EP007: 심법 적용
- **시작**: 마나 수련 중
- **핵심 이벤트**:
  1. 혈영심법을 마나에 적용
  2. 시행착오 반복
  3. 카이든의 응원
  4. 미세한 진전
- **끝**: 가능성 확인
- **등장인물**: 무영, 카이든

#### EP008: 시련
- **시작**: 수련 지속
- **핵심 이벤트**:
  1. 의뢰 중 위기
  2. 마나 제어 불안정
  3. 부상
  4. 좌절하지 않고 다시 도전
- **끝**: 무너지지 않는 무영
- **등장인물**: 무영

#### EP009: 각성의 실마리
- **시작**: 부상 회복 중
- **핵심 이벤트**:
  1. 명상 중 깨달음
  2. 마나와 내공의 융합점 발견
  3. 심법 변형
  4. 마나가 움직이기 시작
- **끝**: 각성 직전
- **캐릭터 경지**: 마나 순환 시작
- **등장인물**: 무영

#### EP010: 그래듀에이트 (1부 마무리)
- **시작**: 각성 직전 상태
- **핵심 이벤트**:
  1. 완전한 마나 각성
  2. 전투에서 실력 발휘
  3. 주변이 놀람
  4. "검은 별"이라는 별명 시작
- **끝**: 그래듀에이트 진입, 새로운 시작
- **캐릭터 경지**: **그래듀에이트**
- **등장인물**: 무영, 카이든

## 2부: 검은 별 (11~20화)

### 핵심 테마
- 빠른 성장과 주목
- 에이라와의 만남
- 소드마스터 등극

### 에피소드별 플롯

#### EP011: 비정상적 성장
- **핵심 이벤트**:
  1. 성장 속도가 비정상적
  2. 길드에서 주목받기 시작
  3. 소문이 퍼짐
- **등장인물**: 무영, 카이든

#### EP012: 에이라
- **핵심 이벤트**:
  1. 에이라와 첫 만남
  2. 서로 경계
  3. 둘 다 아웃사이더임을 인지
- **등장인물**: 무영, 에이라
- **에이라 경지**: 6서클

#### EP013: 공감
- **핵심 이벤트**:
  1. 에이라의 배경 (하프엘프 차별)
  2. 무영의 고독과 공명
  3. 서로를 이해하기 시작
- **등장인물**: 무영, 에이라

#### EP014: 교환
- **핵심 이벤트**:
  1. 마법 ↔ 심법 교환 제안
  2. 에이라가 마법 설명
  3. 무영이 심법 원리 설명
  4. 협력 시작
- **등장인물**: 무영, 에이라

#### EP015: 볼드릭
- **핵심 이벤트**:
  1. 좋은 검이 필요함
  2. 드워프 대장장이 볼드릭 소개
  3. 검 제작 의뢰
  4. 볼드릭의 까칠함
- **등장인물**: 무영, 볼드릭

#### EP016: 던전
- **핵심 이벤트**:
  1. 어려운 던전 공략
  2. 에이라와 협력
  3. 명성 상승
- **등장인물**: 무영, 에이라

#### EP017: 견제
- **핵심 이벤트**:
  1. 기존 강자들의 견제
  2. 시기와 의심
  3. 무영은 신경 쓰지 않음
- **등장인물**: 무영

#### EP018: 협력
- **핵심 이벤트**:
  1. 위기 상황 발생
  2. 에이라와 본격 협력
  3. 서로의 실력 인정
- **등장인물**: 무영, 에이라

#### EP019: 새 검
- **핵심 이벤트**:
  1. 볼드릭의 검 완성
  2. 검의 진가
  3. 실력 급상승
- **등장인물**: 무영, 볼드릭

#### EP020: 소드마스터 (2부 마무리)
- **핵심 이벤트**:
  1. 대결에서 소드마스터 등극
  2. "검은 별" 별명 확정
  3. 대륙에 이름이 퍼짐
- **캐릭터 경지**: **소드마스터**
- **에이라 경지**: 7서클
- **등장인물**: 무영, 에이라, 카이든

## 3부: 용의 친구 (21~30화)

### 핵심 테마
- 명성 확산
- 이그니스와의 우정
- 마왕의 존재 인지

### 주요 이벤트
- 21화: 제국에서 관심
- 22화: 이그니스 첫 조우
- 23-26화: 이그니스와 친구가 됨
- 27화: 레인 등장, 정보 수집
- 28-29화: 북방의 이상한 움직임
- 30화: "마왕"의 존재 확인

---

## 4부: 대륙의 그림자 (31~40화)

### 핵심 테마
- 혈마의 정체 확인
- 정치적 소용돌이
- 전면전 불가피

### 주요 이벤트
- 31-32화: 마왕에 대한 정보 수집
- 33화: **혈마임을 확인** (핵심 반전)
- 34화: 무영의 분노와 결심
- 35-36화: 제국/왕국 정치
- 37화: 에이라 과거 청산
- 38화: 카이든 그래듀에이트 달성
- 39-40화: 마왕군 첫 교전

---

## 5부: 전쟁의 서막 (41~50화)

### 핵심 테마
- 대륙 연합군
- 동료들의 성장
- 최종전 준비

### 주요 이벤트
- 41-43화: 연합군 결성, 무영이 핵심으로
- 44화: 에이라 8서클 돌파
- 45-46화: 첫 대규모 전투
- 47화: 카이든 소드마스터 각성
- 48화: 에이라 9서클 (대마법사)
- 49-50화: 마왕성 진격 결정

---

## 6부: 혈영, 다시 (51~60화)

### 핵심 테마
- 마왕성 공략
- 최종 대결
- 무림 귀환

### 주요 이벤트
- 51-52화: 마왕성 진격 시작
- 53화: 이그니스 vs 마왕군 용
- 54화: 에이라 vs 마왕군 대마법사
- 55화: 카이든 vs 마왕군 장수
- 56-57화: 무영, 혈마와 재회
- 58화: **무영 vs 혈마 최종 대결**
- 59화: 그랜드 소드마스터 돌파, 혈마 처치
- 60화: **무림 귀환, 설하와 재회**
