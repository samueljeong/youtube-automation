# 코드 정리 (Code Simplifier)

작업 완료 후 코드를 정리하고 최적화합니다.

## 사용법

파일 경로를 인자로 전달: `/simplify drama_server.py`

## 정리 항목

### 1. 불필요한 코드 제거
- 사용하지 않는 import 제거
- 주석 처리된 코드 삭제
- 중복 코드 통합

### 2. 코드 품질 개선
- 함수/변수명 명확화
- 매직 넘버를 상수로 추출
- 긴 함수 분리 (50줄 이상)

### 3. 성능 최적화
- 함수 내부 import → 파일 상단으로 이동
- 불필요한 루프 최적화
- 리스트 컴프리헨션 활용

## 실행 명령

```bash
# 린터로 자동 수정
ruff check $ARGUMENTS --fix --unsafe-fixes

# 포맷팅
ruff format $ARGUMENTS
```

## 체크리스트

- [ ] 사용하지 않는 import 제거
- [ ] 중복 코드 없음
- [ ] 함수 길이 적절 (50줄 이하)
- [ ] 명확한 변수명
- [ ] 린터 경고 없음
